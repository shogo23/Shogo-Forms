-- Description --
Shogo Forms is a form submission for worpdress websites and blogs. User can customize any form field they want to display in the
front end and with custom field layout to match the layout of the websites and blog. Shogo Forms can also embed to your theme widgets.

-- Installation --
1. Upload the entire shogo-forms directory to your wordpress plugin directory "/wp-content/plugins"
2. Login to your Wordpress Admin Panel and activate the plugin through the plugins menu. "www.your-wordpress-website.com/wp-admin/plugins.php"
3. A Shogo Forms Link will appear in your wordpress menu at the bottom. Click the link and create form to start.

On Github
1. Clone shogo-forms from https://github.com/shogo23/shogo-forms to your wordpress plugins directory.
2. Rename directory to "shogo-forms".
3. Login to your Wordpress Admin Panel and activate the plugin through the plugins menu. "www.your-wordpress-website.com/wp-admin/plugins.php"
4. A Shogo Forms Link will appear in your wordpress menu at the bottom. Click the link and create form to start.

Make Sure /wp-content/uploads/sf_forms directory is rewriteble if you are using linux operating system. This is the directory
storage for file attachments.
Open your command terminal and type "sudo chmod 777 -R /<path to your wordpress>/wp-content/uploads/sf_forms"
